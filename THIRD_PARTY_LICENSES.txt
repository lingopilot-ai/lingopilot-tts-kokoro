4. eSpeak-NG runtime shared library
License: GNU General Public License v3.0
Upstream: https://github.com/espeak-ng/espeak-ng

This sidecar does not link eSpeak-NG at build time. It loads the eSpeak-NG shared library at runtime from the directory passed through `--espeak-data-dir`. Users who receive `espeak-ng.dll` in a packaged release may obtain the corresponding source code from the upstream eSpeak-NG project.
